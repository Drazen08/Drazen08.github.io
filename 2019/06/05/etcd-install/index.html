3.安装etcd

3.1静态配置

ip为80服务器etcd部署

3.1.1 cfssl 工具的安装 ，用于创建各类证书。安装方式可自行百度。
3.1.2 创建目录 /opt /opt/cfssl/etcdSSL
3.1.3 创建 CA 配置文件（ca-config.json）
{
  "signing": {
    "default": {
      "expiry": "876000h"
    },
    "profiles": {
      "etcd": {
        "usages": [
            "signing",
            "key encipherment",
            "server auth",
            "client auth"
        ],
        "expiry": "876000h"
   }
 }
}
}
3.1.4 创建 CA 证书签名请求（ca-csr.json）
3.1.5 生成 CA 证书和私钥
3.1.6 创建 etcd证书签名请求（etcd-csr.json）
3.1.7 生成 etcd证书和私钥
3.1.8 确认生成证书列表 并 创建 /etc/etcd/etcdSSL 目录
3.1.9 将所有证书文件拷贝至 /etc/etcd/etcdSSL 目录
  • cp /opt/cfssl/etcdSSL/* /etc/etcd/etcdSSL
3.1.10 安装 etcd服务
  • yum install -y etcd
3.1.11 配置 etcd 的 service文件（/usr/lib/systemd/system）
  • vim /usr/lib/systemd/system/etcd.service
3.1.12 编写 etcd的配置文件（/etc/etcd/etcd.conf）

这是172.17.34.80节点的配置，如果配置其他etcd节点只要将上面的IP地址改成相应节点的IP地址即可。
